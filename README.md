# Dashboard Capability & Commercial Experience Foundation

This project is a UI/UX dashboard designed in Figma for a commercial
intelligence and operations platform.

## Project Overview

The dashboard includes:

-   Dashboard Overview
-   Lead Management
-   Campaign Workspace
-   Meeting Workspace
-   Customer Workspace
-   Analytics Dashboard
-   Design System
-   Component Library
-   Dashboard Documentation
-   AI Reflection
-   Questions & Risks
-   Deliverables Summary

## Tools Used

-   Figma
-   ChatGPT (Research & Content Support)

## Features

-   Clean and modern dashboard UI
-   Consistent design system
-   Reusable components
-   Data visualization with charts and tables
-   Organized workflow screens
